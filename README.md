## 독일 대학원 검색 시스템
### 📖프로젝트 소개
- MySQL과 장고를 연결하여 **원하는 조건에 맞는 독일의 석사 프로그램**을 검색할 수 있는 웹페이지입니다.
- 원하는 학교를 **즐겨찾기에 추가 및 삭제**하는 것이 가능하고, **새로운 학교를 학교 리스트에 추가하거나 삭제**하는 기능도 갖추고 있습니다.
